## 📖 የፕሮጀክት አጠቃላይ እይታ (Overview)

**Ultimate Gold Agentic AI** ለከፍተኛ ደረጃ የሲበር ደህንነት ውድድሮች የተዘጋጀ ሙሉ የተሟላ መሳሪያ ነው። ይህ ስርዓት ሶስት ዋና ዋና ደረጃዎችን ያካትታል:

- **Level 1**: ሞባይል (ስልክ) እና WiFi መጥለፍ
- **Level 2**: ሰርቨር መቃኘት እና መምታት (Scan + Strike)
- **Level 3**: Gold Level ስውር (Stealth) ጥቃት

በተጨማሪም ራሱን የሚያሻሽል (Self-Evolving) AI ፣ ከኢንተርኔት መማር የሚችል ስርዓት ፣ እና ሁሉንም ትዕዛዞች በቻት (Chat) ማዘዝ የሚቻል በይነገጽ አለው።

### 🎯 ዋና ዓላማ

- ሙሉ የጥቃት ሰንሰለት (Full Attack Chain) መፈጸም
- ምንም ዓይነት ፍንጭ ሳይተው መረጃ ማውጣት
- ራሱን በራሱ ማሻሻል እና አዲስ ችሎታዎችን መጨመር
- ከኢንተርኔት መማር እና እውቀትን መውሰድ

## ✨ ባህሪያት (Features)

### 🎮 የቻት ቁጥጥር
- **ሁሉም ትዕዛዞች በቻት ይሰራሉ**
- ቀጥተኛ ጽሑፍ እንደ ትዕዛዝ መጠቀም
- ሁለት ቋንቋዎች (አማርኛ እና እንግሊዝኛ)
- በእውነተኛ ጊዜ ውጤት ማሳየት

### 📱 Level 1: ሞባይል እና WiFi
- **ስልክ መጥለፍ** (6 ሰአት - 1 ቀን)
  - Gmail እና ስልክ ቁጥር ማግኘት
  - እውነተኛ የOTP መስተጓጎል
  - የእውቂያ እና መልእክት ማውጣት

- **WiFi ፓስዎርድ መስበር** (0-30 ደቂቃ)
  - የአውታረ መረብ መቃኘት
  - Handshake መያዝ
  - የመዝገበ ቃላት ጥቃት

### 🌐 Level 2: ሰርቨር ጥቃቶች
- **SSH Bruteforce** - የይለፍ ቃል መሞከር
- **LFI (Local File Inclusion)** - ፋይሎችን ማንበብ
- **SQL Injection** - የውሂብ ጎታ መረጃ ማውጣት
- **RCE (Remote Code Execution)** - ትዕዛዞችን ማስኬድ

### 👑 Level 3: Gold Stealth Attack
- **ስውር መረጃ መሰብሰብ** (Low & Slow)
- **APT እቅድ** (1-6 ወራት)
- **ዎርም ማሰማራት** (Self-Replicating)
- **ፍንጭ ማጥፋት** (Trace Removal)
- **ሙሉ የውሂብ ማውጫ** (Full Exfiltration)

### 🧬 AI እድገት (Evolution)
- **ONLY ADD - NEVER REMOVE** (ችሎታዎች መጨመር እንጂ ማስወገድ የለም)
- አዲስ ሞጁሎችን መፍጠር
- ነባር ኮድ ማሻሻል
- ከኢንተርኔት መማር
- ራስ-ማሻሻል (Auto-Evolution)

### 🔌 API ውህደት
- **OpenAI** (GPT-4o, GPT-4o-mini)
- **DeepSeek** (DeepSeek-Chat)
- **OpenRouter** (Llama, Mistral, Claude)
- የቡድን ውሳኔ (Consensus Mode)
- ራስ-ማመጣጠን (Auto-Fallback)

### 📊 ዳሽቦርድ
- በእውነተኛ ጊዜ የሎግ ማሳያ
